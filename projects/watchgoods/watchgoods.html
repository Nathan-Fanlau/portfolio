Overview

I developed and designed a website prototype for an online luxury watches collection called WatcHGoods. This project showcased my expertise in both UI/UX design and front-end web development.

My Role

I began by using Figma to create wireframes and interactive prototypes, focusing on intuitive navigation and a responsive layout. Since WatcHGoods centers around high-end, luxury watches, I prioritized creating a sophisticated yet seamless user experience. The design was aimed to balance elegance with simplicity, ensuring the watches were the primary focus of each page.

Development

For the development phase, I used HTML to structure the content, CSS to style it, and JavaScript to handle interactivity and form validations. I implemented mobile responsiveness within the CSS to ensure a consistent user experience across all devices. Finally, I deployed the website on Vercel, making it accessible for public viewing and testing.
