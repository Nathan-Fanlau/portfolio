Overview

MakeMeUpzz is a full-stack web application built using ASP.NET Web Forms and C#, designed to facilitate makeup product transactions. It features role-based access for both admins and users, with secure authentication through session management and cookies. The application allows users to browse and purchase makeup products while administrators manage inventory and orders seamlessly.

My Role

I played a role in both the front-end and back-end of the application. On the back-end, I implemented functionalities for the admin to see transactions history and details, as well as handling transactions made by users. On the front-end, I styled the interface using CSS, ensuring a clean and user-friendly layout. I also built the application following Domain-Driven Design (DDD) principles.
